喜欢玩复古游戏吗？喜欢多人游戏吗？想过与他人实时对战游玩不同的复古游戏吗？

欢迎来到《超级复古游戏之战》！

和朋友一起玩吧，你们可以游玩不同的游戏竞争。你玩游戏得分，不同游戏会以不同方式处理分数。但请注意，他人得分时，你的游戏难度也会提升。

基于经典复古游戏的六种不同的单人和多人游戏版本，还有更多正在开发中！

没有广告。没有游戏内购买。只有复古游戏和美好时光。

欢迎在 GitHub 上提供反馈。

已知限制：

* 需要进行全面的游戏测试，以平衡游戏、找到并修复服务器的缺陷。
* 未经多种屏幕尺寸测试。

请在 GitHub sponsors 或 Liberapay 上支持后续开发。
